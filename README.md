# Смарт-контракт для заключения пари.

Данный проект представляет собой приложение для заключения пари на блокчейне Ethereum, построенное на базе шаблона Scaffold-ETH 2. Приложение имеет простой интерфейс, состоящий из двух основных элементов: формы для создания нового пари и отображения количества активных пари.

В верхней части интерфейса расположена форма "Пари" с полем для ввода названия пари, полем для указания суммы ставки в ETH (по умолчанию 0.01 ETH) и кнопкой "Создать". Ниже отображается заголовок "Активные пари:" с текущим количеством активных пари (в данном случае 0), что означает отсутствие активных пари на данный момент.

Это версия демонстрирует базовые принципы децентрализованных приложений: прямое взаимодействие пользователя со смарт-контрактом через веб-интерфейс, хранение данных в блокчейне и отсутствие централизованного сервера-посредника.
