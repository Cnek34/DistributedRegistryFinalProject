# Смарт-контракт для заключения пари.

## Основные возможности

- Создание пари с указанием:
  - названия
  - стороны («за» / «против»)
  - суммы ставки
  - времени жизни пари
- Отображение списка активных пари
- Просмотр информации по каждому пари:
  - название
  - кто создал
  - сумма ставки
  - выбранная сторона
- Прямое взаимодействие с блокчейном Ethereum
